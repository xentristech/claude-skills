# SEO On-page — Contenidos (Bloque 2 y 3)

Para cada URL a crear u optimizar, completa estos campos siguiendo los criterios.

## URL
- Es legible y corta.
- Tiene máximo 5 palabras.
- Da contexto del contenido.

## Término de búsqueda principal
- La búsqueda principal a posicionar (1 keyword objetivo por URL).

## Términos de búsqueda secundarios
- Búsquedas secundarias que corresponden al mismo objetivo/intención de búsqueda.

## Formato de contenido
- Identificar la intención de búsqueda de las personas.
- Definir el formato ideal para llegar al top de posiciones (landing, guía, comparativa,
  caso de estudio, listado, etc.).

## Título SEO
- Ofrece una solución a la búsqueda de la persona.
- Usa de manera natural los términos de búsqueda objetivo.
- Título único para cada página del sitio web.
- Verificar extensión → https://mangools.com/free-seo-tools/serp-simulator

## Meta descripción
- Describe lo que la persona encontrará en el contenido.
- Usa las búsquedas de las personas como lenguaje principal para describir el contenido.
- Texto descriptivo único para cada página del sitio web.
- Verificar extensión → https://mangools.com/free-seo-tools/serp-simulator

## Título principal del contenido (H1)
- Da el contexto de todo el contenido.
- Usa el lenguaje de búsqueda de las personas.
- Es un texto único para cada página.

## Recomendaciones de estructura de contenido
- Párrafos cortos
- Listas
- Secciones
- Negritas
- Imágenes
- Multimedia

---

## Bloque 3 — Creación de contenidos editorial

Mismos criterios de Título SEO y Meta descripción aplicados a piezas editoriales
(blog/vlog, casos de estudio, guías). Cada pieza debe responder a una intención de
búsqueda concreta y enlazar internamente con las páginas comerciales relevantes.
